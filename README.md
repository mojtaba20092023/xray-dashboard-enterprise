# 📊 Xray Enterprise Dashboard

داشبورد زنده و پیشرفته برای مانیتورینگ Xray با قابلیت‌های کامل:

- نمایش وضعیت تست‌ها به‌صورت لحظه‌ای (Real-time)
- نمودار میله‌ای و خطی برای زمان پاسخ‌دهی
- ذخیره‌ی نتایج در پایگاه داده (PostgreSQL یا MongoDB)
- هشدار مرورگر + هشدار صوتی
- ارسال ایمیل و پیام تلگرام هنگام خطا
